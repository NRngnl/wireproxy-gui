# Security Policy

## Supported Versions

Security updates are provided for the latest released version.

## Reporting A Vulnerability

Please report security vulnerabilities through GitHub private vulnerability
reporting on this repository. Do not open a public issue for a suspected
vulnerability.

Include:

- the affected version or commit,
- the platform you tested on,
- a clear reproduction path,
- the expected impact,
- whether the issue affects WireGuard profiles, Tailscale profiles, or both.

I will acknowledge valid reports and coordinate a fix before public disclosure.
